多用户跨设备非局域网登陆

1 支持内网穿透 https://www.zerotier.com/
2 支持多用户登陆 https://github.com/anhkgg/SuperRDP
3 支持跨设备连接 Microsoft Remote Desktop

配置内网穿透以实现异地连接

⏬ 下载并安装zerotier

📒 注册zerotier账号，创建zerotier私有网络

➕ 打开zerotiger软件，使用私有网络ID加入

☑️ 勾选设备注册，打开桥接设置（Allow Ethernet Bridging）

📝 记录对应设备的IP地址（不是物理IP）

![[Pasted image 20221002001835.png]]

Windows端多用户设置

⏬ 下载并安装SuperRDP （安装时系统可能会警告有病毒，需转到Windows安全中心允许该文件）

🔛 打开SuperRDP，勾选自动分析和开机启动（需要关注公众号获得使用码），安装SuperRDP服务

![[Pasted image 20221002002025.png]]

🆕 新建Windows账户，设置–账户–家庭和其他用户–其他用户-添加（没有此人信息）按提示操作完成账户新建，可以更改账户类型为管理员，以使用一些软件

使用Windows Remote Desktop连接至用户

目前仅测试Mac 及 Win，下载链接remote-desktop-clients
Mac & Win，下载并安装Remote Desktop软件
Ubuntu
sudo apt install xrdp && sudo systemctl enable --now xrdp
sudo ufw allow from any to any port 3389 proto tcp # open a firewall port 3389 for an incoming traffic

➕ 打开Windows远程控制功能，此电脑–属性-远程桌面-启用远程桌面

🔗 打开Windows Remote Desktop，使用📝Zerotiger中目标机器的IP地址以及所需登陆的账户密码，即可实现多用户同时使用一台Windows电脑了。

➕ 如果Windows新用户下部分软件无法使用，可以右键以管理员身份运行。
